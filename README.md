# Caderno-acoes-focadas-em-dividendo
# Análise de Estratégias de Investimento: Ações Focadas em Dividendos

Repositório destinado ao armazenamento de material acadêmico sobre políticas de distribuição de proventos e análise fundamentalista de companhias listadas na B3, com ênfase em Dividend Yield e Payout.

## 📊 Objetivo do Projeto

Este projeto tem como finalidade compilar e analisar dados acerca de companhias brasileiras que possuem histórico de distribuição consistente de dividendos e Juros sobre Capital Próprio (JCP). O estudo busca relacionar os indicadores de proventos com métricas contábeis como Lucro por Ação (LPA), Retorno sobre o Patrimônio Líquido (ROE) e Fluxo de Caixa Livre.
